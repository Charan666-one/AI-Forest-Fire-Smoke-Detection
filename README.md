AI-Based Forest Fire & Smoke Detection
Overview

I built this project to detect forest fire and smoke regions using machine learning. The goal was to understand how effectively we can identify high-risk areas from aerial image-based features and use it for early warning.

Features

Implemented a Random Forest model for classification

Evaluated the model using Confusion Matrix and ROC Curve

Created a fire risk heatmap for better visualization

Results

Accuracy: 93%

ROC-AUC Score: 0.97

Tech Stack

Python

Scikit-learn

Pandas, NumPy
